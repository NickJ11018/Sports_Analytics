Feature Engineering¶

We have a lot of data, mostly categorical, and numerical features that appear to at least be reasonably distributed, so we opted to forego further cleaning and focus on better model tuning for now.

We did feel that the price of an individual item for each rep was an interesting data point, however, given that it is easily calculated by simple division, it felt inefficient to add that to the data. Instead, we opted for the difference between each sales rep's average price and the overall average price for a given product, which we called Price_dev.

In [306]:
df['Each'] = df['Val'] / df['Quant']
df['Avg_Each'] = df.groupby('Prod')['Each'].transform('mean')
df['Price_dev'] = df['Each'] - df['Avg_Each']

Categorical variables need to be converted to "dummy" columns, one for each category. A large number of categorical variables (5080 ID + 797 Prod) means a lot of dummy variables.

Since most of these will be zeroes, we can use truncated SVD, a dimension reduction technique that works particularly well for sparse data.

Optimizing SVD means figuring out how many components you need to capture the majority of the variance in your data.

The plot of the number of components v. the cumulative captured variance is called a Scree Plot.

As you can see below, we are only getting about 18% of our variance at 50 components.

We predetermined how much of the data would be labeled anomalous when we first made the model, with a parameter called contamination. The model gives every point a score, and all this parameter does is decide where the line between the two categories is.

Since we have test data, we can set the contamination value to what we expect it to be. This isn't really data leakage because all it controls is the raw number of anomalous values. If our model is not very good, all an inflated contamination value will do is give us more wrong answers.

Big Modeling¶

Now we are ready to see how our model performs by training it, then running unseen data through it and seeing it we find the same anomalies. We will be using F1 score to evaluate the models.

We built a big class to make this easier.

In [343]:
class AnomalyPipeline:
    def __init__(self, n_components=1000, n_estimators=100, contamination=0.08, random_state=42):
        self.n_components = n_components
        self.n_estimators = n_estimators
        self.contamination = contamination
        self.random_state = random_state

        self.scaler = None
        self.svd = None
        self.model = None
        self.num_cols = ['Quant', 'Val', 'Price_dev']
        self.extra_cols = ['Each', 'Avg_Each', 'Insp', 'anomaly', 'anomaly_label']
        self.dummy_columns = None  # Will be tracked to align test data

    def preprocess(self, df, drop_nulls=True, fit=True):
        df = df.copy()
        if drop_nulls:
            df = df.dropna()
        df['Each'] = df['Val'] / df['Quant']
        df['Avg_Each'] = df.groupby('Prod')['Each'].transform('mean')
        df['Price_dev'] = df['Each'] - df['Avg_Each']

        # Numeric features
        X_num = df[self.num_cols]
        if fit:
            self.scaler = StandardScaler()
            X_num_scaled = self.scaler.fit_transform(X_num)
        else:
            X_num_scaled = self.scaler.transform(X_num)

        # Categorical dummies
        df_dummies = pd.get_dummies(df.drop(columns=self.num_cols + self.extra_cols), drop_first=True)

        X_sparse = csr_matrix(df_dummies.values)

        return X_num_scaled, X_sparse

    def fit(self, df_train):
        X_num_scaled, X_sparse = self.preprocess(df_train, fit=True)

        self.svd = TruncatedSVD(n_components=self.n_components, random_state=self.random_state)
        X_dummies_reduced = self.svd.fit_transform(X_sparse)

        X_final = np.hstack([X_num_scaled, X_dummies_reduced])

        self.model = IsolationForest(
            n_estimators=self.n_estimators,
            contamination=self.contamination,
            n_jobs=-1,
            random_state=self.random_state
        )
        self.model.fit(X_final)
        return self

    def predict(self, df_test):
        X_num_scaled, X_sparse = self.preprocess(df_test, fit=False)
        X_dummies_reduced = self.svd.fit_transform(X_sparse)
        X_final = np.hstack([X_num_scaled, X_dummies_reduced])

        labels = self.model.predict(X_final)
        scores = self.model.decision_function(X_final)

        return labels, scores
