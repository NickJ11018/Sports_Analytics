# Sports_Analytics
Compilation of all the analytics work I've done, relatively unstructured

This contains a "The Best" folder for the main highlights for anyone interested. Included there are the proprietary MasterDoc I created for a first shot at maintaining position-specific predictions and records for the MLR. This uses Excel's Solver add-in for the bulk of the calculations. Along with it is the StatsOnTap documents which are SQL code blocks I wrote to do some research and ad-hoc queries, as well as develop the FFF system. There are a couple early explorations of the FFF structure I completed in an earlier class. 

Additionally, there is a recent project optimizing the Baltimore Orioles's free agency acquisitions by maximizing WAR while controlling for the salary cap considerations.

The main projects have all been through SQL or Excel, looking at fantasy projections for Major League Rugby players as well as a proprietary "Friendly Fantasy Format" method of introducing new fantasy managers and predicting game outcomes.
